# 📊 Student Mental Health Insights Dashboard

An interactive **data-driven dashboard** built with **Python, MySQL, and Streamlit** to analyze student mental health trends and uncover gaps in help-seeking behavior.

---

## 🚀 Overview

This project transforms raw student survey data into **interactive insights** by combining:

* SQL-based data querying
* Python data processing
* Streamlit dashboard UI
* Plotly visualizations

It focuses on three key mental health indicators:

* Depression
* Anxiety
* Panic Attacks

👉 The goal is **identifying meaningful patterns and gaps in support systems**.

---

## 🧠 Key Insight

> Only a small percentage of students experiencing mental health issues seek professional help.

This highlights a **critical disconnect between need and action**, showing that the problem is behavioral and systemic.

---

## ✨ Features

### 📌 1. KPI Dashboard

* Total students at risk
* Students who sought help
* Help-seeking rate (%)

---

### 📊 2. Interactive Visualizations

* **Pie Chart** → Distribution of selected mental health condition (Yes vs No)
* **Bar Chart** → Breakdown by:

  * Course
  * Gender
  * Year of Study

---

### 🎛️ 3. Dynamic Filters (Sidebar)

* Select mental health indicator
* Compare across different student groups

---

### 📈 4. Automated Insights

* Identifies **most affected group dynamically**
* Highlights patterns across demographics

## ⚙️ How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/JK3690/Student-Mental-Health-Analysis.git
cd Student-Mental-Health-Analysis
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
